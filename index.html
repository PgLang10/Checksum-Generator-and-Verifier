Learn ‚Äî Internet Checksum Calculation

The Internet Checksum is a technique used to detect errors in network transmission. It uses binary addition and 1‚Äôs complement arithmetic to verify data integrity in protocols like TCP, UDP, and ICMP.

Steps of Checksum Calculation

  1. Step 1: Convert the message into binary format. Each character is represented by its ASCII code in binary.
  2. Step 2: Divide the binary string into fixed-size words (8-bit or 16-bit).
  3. Step 3: Perform binary addition (1‚Äôs complement addition) of all words. If a carry bit is produced, wrap it around and add it back to the least significant bit.
  4. Step 4: Take the 1‚Äôs complement of the final sum to form the checksum.
  5. Step 5: At the receiver, add all words including the checksum. If the final result is all 1‚Äôs, the message is error-free; otherwise, it indicates corruption.

Example

Consider two 8-bit words:
Word 1 = 01010101
Word 2 = 11001100
Sum = 01010101 + 11001100 = 1 00100001
Wrap-around carry: 00100001 + 1 = 00100010
1‚Äôs Complement = 11011101 ‚Üí Checksum = 11011101

Key Concept

The checksum ensures that even a single-bit change during transmission alters the final sum, allowing the receiver to detect that an error has occurred.
